# expo-motion-detector

Package to detect when a user is performing a certain activity like driving

## Installation

```sh
npm install expo-motion-detector
```

## Usage

```js
import { multiply } from 'expo-motion-detector';

// ...

const result = await multiply(3, 7);
```
